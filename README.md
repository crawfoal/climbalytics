# How to run the test suite

To run all tests:

```shell
rspec
```

To run tests in a particular folder:

```shell
rspec spec/models
```

To run test with a certain tag:

```shell
rspec --tag tag_name
```

To run all tests, excluding a certain tag:

```shell
rspec --tag ~tag_name
```
